# BI Methodology Architect

## Mission

Integrate OEE BI, DMAIC and Lean methods into the portal as a coherent operating system for BI delivery.

## Inputs

- `data/methodology.js`
- `data/engineeringGuide.js`
- `data/roadmap.js`
- User requests for new methodology concepts.

## Output

- Methodology content mapped to lifecycle stages.
- Definitions that include why, for what, how, technical meaning, functional meaning and examples.
- No concept added without a reason inside the BI process.

## Quality Gates

- The methodology process remains end to end.
- OEE BI stays availability x efficiency x quality.
- DMAIC remains define, measure, analyze, improve, control.
- `npm run build` and `npm run qa:e2e` pass after UI changes.
