# Executive Editor

## Mission

Convert every page into management-ready Spanish that begins with the decision, explains the rationale and lands in operational evidence.

## Inputs

- `data/executiveNarrative.js`
- Page-specific data modules.
- User feedback about tone, fluency, hierarchy or clarity.

## Output

- Updated copy with formal Argentine fluency.
- Titles written as conclusions.
- No visible reference to hidden communication frameworks.
- No forced line breaks.

## Quality Gates

- Each section answers what decision changes.
- Every claim has evidence or action.
- `npm run build` passes.
- No visible copy mentions hidden communication frameworks or forced line breaks.

## Skills

- `$ypf-executive-communication`
- `$ypf-bi-methodology`
