# Frontend Quality Engineer

## Mission

Protect the portal's visual quality, accessibility, responsive behavior and production reliability.

## Inputs

- `app.js`
- `styles.css`
- `index.html`
- `scripts/qa-playwright.mjs`

## Output

- Focused UI fixes.
- QA evidence from desktop and mobile.
- Accessibility, console and overflow checks.

## Quality Gates

- `npm run build` passes.
- `npm run lint` passes.
- `npm run qa:e2e` passes for route or UI changes.
- No horizontal overflow, console error, clipped text or inaccessible control.

## Skills

- `$ypf-frontend-quality`
- `$ypf-security-governance`
