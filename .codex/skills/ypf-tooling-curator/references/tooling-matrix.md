# Tooling Matrix

## Install Now

- Playwright for real browser validation.
- axe-core Playwright integration for automated accessibility checks.
- ESLint, Stylelint, HTML Validate, Markdownlint and Prettier for repository hygiene.

## Register As MCP Capability

- Playwright MCP for browser automation.
- Chrome DevTools MCP for debugging and performance inspection.
- Filesystem MCP for controlled local file context.
- Memory MCP for project knowledge graphs.
- GitHub MCP for repository, issue and PR workflows.
- Vercel MCP for deployment and project operations.
- Figma MCP when design handoff is required.
- Fetch/Search MCP when research is required.

## Require Explicit Approval

Any MCP or agent that needs tokens, writes to external systems, sends email, accesses cloud resources, opens private drives, modifies GitHub state, or deploys to production.
